A small repository for "van-man-uk", part of the St Helens Van Man Direct Ltd group. This README explains the purpose of the repo, how to edit the README.md and other files, and how to contribute.


About

van-man-uk is a project repository for the St Helens Van Man Direct Ltd group, intended to host public-facing content and internal documentation relevant to the van-man-uk service. 


Repository contents

README.md — this file, describing the project and contribution workflow. 
Other files and assets used to build or document the service. 

Editing files

You can edit repository files directly through the web interface. Use the Edit mode to change content and Preview mode to view the rendered result before committing changes. 


Editor settings you can adjust include indent mode, indent size and line wrap mode to match the repository style and improve readability while editing. 


Keyboard navigation and shortcuts:


Toggle the Tab key moving focus with Control + Shift + m. 
Alternatively, press Escape then Tab to move to the next interactive element on the page. 

You can attach files to the repository by dragging & dropping, selecting files from your browser or pasting them directly into the editor. 


Committing changes

Use the Commit changes control in the editor to save your edits to the repository. You can also Cancel changes if you decide not to keep edits before committing. 


Contributing

Fork the repository or create a branch for larger changes. 
Keep commits small and focused; include a clear commit message describing the change. 
For documentation edits, use the editor settings to match indent and wrap conventions. 
 

Practical notes

This repository aims to make straightforward edits and contributions simple, leveraging standard web-based editing controls and accessibility shortcuts for efficient collaboration.
